# Line Follower Robot

Ένα μικρό ρομπότ που ακολουθεί μαύρη γραμμή πάνω σε λευκή επιφάνεια. Φτιάχτηκε με Raspberry Pi Pico, τρεις αισθητήρες υπερύθρων και έναν απλό αλγόριθμο PID για να κινείται όσο πιο ομαλά γίνεται.


## Τι χρειάστηκε

- Raspberry Pi Pico
- L298N (για να ελέγχει τους κινητήρες)
- 2 DC κινητήρες με τροχούς
- 3 αισθητήρες γραμμής TCRT5000
- 4 μπαταρίες AA
- Μετατροπέας τάσης LM2596
- Πλαίσιο ρομπότ 2WD
- Καλώδια και breadboard


## Πώς είναι συνδεδεμένο

### Κινητήρες
L298N
IN1 | GP0
IN2 | GP1
IN3 | GP2
IN4 | GP3
ENA | GP4
ENB | GP5

### Αισθητήρες
| Αισθητήρας | Pico |
| Αριστερός | GP10 |
| Κεντρικός | GP11 |
| Δεξιός | GP12 |

> Τροφοδοτούνται με 3.3V από το Pico

### Τροφοδοσία
Οι μπαταρίες τροφοδοτούν τους κινητήρες μέσω του L298N και το Pico μέσω του LM2596 που κατεβάζει την τάση στα 5V. Όλα μοιράζονται κοινή γείωση.

## Πώς λειτουργεί

Οι τρεις αισθητήρες κοιτάνε κάτω και ανιχνεύουν αν βλέπουν μαύρο ή άσπρο. Ανάλογα με το τι βλέπουν, ο αλγόριθμος PID υπολογίζει πόσο έχει ξεφύγει το ρομπότ από τη γραμμή και διορθώνει την ταχύτητα των τροχών.

- Αν βλέπει μόνο ο κεντρικός → πάει ευθεία
- Αν βλέπει και ο αριστερός → στρίβει αριστερά
- Αν βλέπει και ο δεξιός → στρίβει δεξιά
- Αν χάσει τη γραμμή → ψάχνει μόνο του
- Αν όλοι δουν μαύρο → σταματά (τέλος πίστας)

## Για να το τρέξεις

1. Ανέβασε το `main.py` στο Pico μέσω Thonny
2. Βάλε το ρομπότ στην αρχή της γραμμής
4. Απόλαυσε 🎉

---

## Ρυθμίσεις PID

Αν θέλεις να πειραματιστείς με την ταχύτητα ή τη συμπεριφορά του ρομπότ:

```python
BASE_SPEED = 65000  # πόσο γρήγορο πάει
KP = 30000          # πόσο απότομα στρίβει
KI = 0              # άφησέ το στο 0
KD = 15000          # πόσο ομαλά μπαίνει στις στροφές
```
